# JAAM Fusion 5.x

Репозиторій містить файли прошивки **JAAM Fusion** (кодова назва нової версії) - прошивки для ESP32, що дозволяє за допомогою розміщених на мапі України адресних світлодіодів відображати такі типи інформації:
- 🚨 Повітряні тривоги в реальному часі
- 🌡️ Погода в регіонах України
- ☢️ Радіаційне забруднення
- ⚡ Стан електромережі
- 🇺🇦 Візуальні зображення (прапор України та інші)
- 💡 Режим настільної лампи

Крім цього, підтримується окремий OLED-дисплей, який відображає поточний час, погоду, радіацію, стан електромережі, глобальні сповіщення та сервісні повідомлення.

## 🎯 Що нового в JAAM Fusion 5.x

### 🌟 Головна фішка: Кастомний маппінг LED
**Революційна можливість** налаштування будь-якої апаратної конфігурації світлодіодів!

- **Повна свобода конфігурації**: Підтримка будь-якої кількості та розташування LED
- **Редактор карти LED**: Зручний web-інтерфейс для прив'язки LED до регіонів
- **Гнучкість**: Від простої стрічки до складних матричних конфігурацій
- **Збереження конфігурації**: Маппінг зберігається в пам'яті та відновлюється після перезавантаження
- **Підтримка різних форматів**: Можливість використання різних типів LED (WS2812B тощо) в одному проєкті

Тепер ви не обмежені стандартними конфігураціями - створюйте власні унікальні карти!

### Архітектурні покращення
- **Модульна структура коду**: Кожен компонент системи винесено в окремий клас
- **Асинхронна обробка**: Використання ArduinoAsync та FreeRTOS для ефективної багатозадачності
- **Оптимізація пам'яті**: Покращене управління heap memory та дефрагментація
- **WebSocket протокол**: Real-time оновлення через власний сервер [jaam.net.ua](http://jaam.net.ua/)

### Підтримка нового обладнання
- **ESP32-S3** - повна підтримка з оптимізацією під 4MB Flash
- **ESP32-C3** - підтримка компактних плат
- **Матричні LED конфігурації** - підтримка до 400 LED для великих карт
- **DFPlayer PRO** - відтворення високоякісних аудіофайлів (замість buzzer мелодій)

### Web-інтерфейс
- **GZIP компресія assets** - зменшення розміру веб-файлів на 73.7%
- **Real-time API через WebSocket** - швидкий обмін даними з клієнтами
- **Динамічне стиснення JSON/HTML** - економія трафіку та RAM
- **ETag кешування** - мінімізація повторних завантажень статичних ресурсів
- **Редактор карти та кольорів** - зручне налаштування LED

### Розширена функціональність
- **Нові типи анімацій**: 
  - Fade (згасання)
  - Blink (блимання)
  - Blend Fade (плавний перехід)
  - Pulse (пульсація)
  - One Way Blend Fade (плавний перехід в одному напрямку)
  
- **Покращена підтримка сенсорів**:
  - BME280 (температура, вологість, тиск)
  - BMP280 (температура, тиск)
  - SHT2x, SHT3x, AHT30 (температура та вологість)
  - BH1750 (цифровий сенсор освітлення)

- **Flexible Display Support**:
  - SSD1306 (128x32, 128x64)
  - SH1106G (128x64)
  - SH1107 (128x64)
  - Підтримка обертання дисплея (0°, 90°, 180°, 270°)

### Система оновлень
- **OTA Updates** - оновлення "по повітрю" через веб-інтерфейс
- **Канали розповсюдження**: Production (стабільні) та Beta (експериментальні)
- **Firmware validation** - перевірка цілісності прошивки перед встановленням

## 💻 Програмне забезпечення

### Основні технології
- **Platform**: ESP32 (Espressif32 @ 6.12.0)
- **Framework**: Arduino
- **Async Processing**: ArduinoAsync для non-blocking операцій
- **WebSocket**: Real-time комунікація з сервером
- **LED Control**: Adafruit NeoPixel
- **NTP**: Синхронізація часу
- **Display**: U8g2 для OLED дисплеїв

### Режими роботи мапи

#### 1. Режим тривог
Відображення повітряних тривог на базі офіційного API [ukrainealarm.com](https://www.ukrainealarm.com/) та сторонніх інтеграцій

**Офіційне API:**
- Стандартна тривога (червоний)
- Ракетна небезпека (фіолетовий)
- Загроза БПЛА (помаранчевий)

**e-Tryvoga:**
- Ракетна небезпека (фіолетовий)
- Загроза БПЛА (помаранчевий)
- Загроза КАБів (жовтий)
- Балістична загроза (блакитний)
- Вибухи в регіонах (білий спалах)
- БПЛА-розвідники (синій спалах)

**Спеціальні функції:**
- Виділення домашнього регіону
- Підсвічування нових тривог (інший колір перші N секунд)
- Підсвічування відбою тривоги (інший колір перші N секунд)

#### 2. Режим погоди
Відображення температури за даними [OpenWeatherMap](https://openweathermap.org/)
- Кольорова індикація температури
- Підтримка різних температурних шкал

#### 3. Режим прапора України
Відображення синьо-жовтого прапора України

#### 4. Режим настільної лампи
- Довільні кольори для всіх регіонів
- Підтримка кастомних палітр
- Color picker в web-інтерфейсі

#### 5. Offline режим
Мапа не відображає нічого (LED вимкнені)

### Режими дисплея

#### Годинник
- Великий чіткий час
- Дата
- Синхронізація через NTP
- Підтримка різних часових поясів

#### Погода
- Температура у домашньому регіоні
- Дані з OpenWeatherMap
- Автоматичне оновлення

#### Технічна інформація
- Uptime пристрою
- Використання пам'яті
- Сила WiFi сигналу
- Статус WebSocket з'єднання
- Температура CPU

#### Мікроклімат (при наявності датчика)
- Температура в кімнаті
- Вологість
- Атмосферний тиск (для BME280)

**Приклади стандартних конфігурацій:**
- JAAM 1.3: 26 LED (Київ окремо)
- JAAM 2.1: 26 LED (класична)
- JAAM 3.2: 405 LED (матрична, з підтримкою районів та міст)

**Як працює кастомний маппінг:**
1. Підключіть LED стрічку будь-якої довжини
2. Відкрийте візуальний редактор в web-інтерфейсі

4. Збережіть конфігурацію - вона зберігається в пам'яті

**Переваги кастомного маппінгу:**
- Створення мапи з нестандартною кількістю LED

## 🏠 Інтеграція з Home Assistant

JAAM Fusion повністю інтегрується в Home Assistant через спеціальну **HACS інтеграцію**.

### Встановлення інтеграції

1. **Установіть HACS** (якщо ще не встановлено) - [hacs.xyz](https://hacs.xyz/)
2. **Додати репозиторій інтеграції:**
   - Відкрийте HACS в Home Assistant
   - Перейдіть до "Інтеграції" (Integrations)
   - Натисніть на "🔵 Custom repositories"
   - Додайте: `https://github.com/J-A-A-M/jaam_ha`
   - Категорія: "Integration"
3. **Завантажте інтеграцію:**
   - Знайдіть "JAAM" в HACS
   - Натисніть "Download"
4. **Перезавантажте Home Assistant**
5. **Додайте інтеграцію:**
   - Налаштування → Пристрої та служби → Інтеграції
   - Натисніть "+ Додати інтеграцію"
   - Знайдіть та оберіть "JAAM"
   - Вкажіть IP-адресу вашого JAAM пристрою

**Посилання на інтеграцію:** [github.com/J-A-A-M/jaam_ha](https://github.com/J-A-A-M/jaam_ha)

### Доступні сутності:

**Select (Вибір):**
- Режим мапи (тривоги, погода, прапор, лампа, offline)
- Домашній регіон (вибір з усіх областей та районів України через сервіс HA)

**Light (Освітлення - для режиму лампи):**
- Колір лампи (RGB picker)
- Яскравість лампи (0-100%)

**Sensor (Сенсори):**
- **Системна інформація:**
  - Температура CPU (°C)
  - Uptime пристрою
  - WiFi uptime
  - Сила WiFi сигналу (dBm)
  - Використання пам'яті (bytes)
  - Статус WebSocket підключення
  - WebSocket uptime
  
- **Інформація про тривоги:**
  - Прапорці тривоги в домашньому регіоні
  - Температура в домашньому регіоні (з OpenWeatherMap)

- **Сенсори (якщо підключені):**
  - Кліматичні дані (температура, вологість, тиск з BME280/SHT3x/інших)
  - Рівень освітлення (з BH1750 або фоторезистора)

**Update (Оновлення):**
- Версія прошивки (поточна та доступна)
- Можливість оновлення через HA одним кліком

> **Примітка:** Інтеграція підключається через WebSocket API на порту 81. Всі налаштування яскравості, дисплея, кнопок та інші параметри доступні через веб-інтерфейс пристрою (за IP адресою або mDNS ім'ям).

### Вимоги
- [PlatformIO](https://platformio.org/) (CLI або VS Code extension)
- Python 3.x (для скриптів компресії)

### Збірка прошивки

```bash
# Клонування репозиторію
git clone https://github.com/J-A-A-M/jaam_fusion.git
cd jaam_fusion

# Збірка для стандартної плати
platformio run -e firmware

# Збірка для ESP32-S3
platformio run -e firmware_esp32s3

# Збірка для ESP32-C3
platformio run -e firmware_esp32c3

# Збірка та завантаження
platformio run -e firmware --target upload

# Моніторинг Serial
platformio device monitor
```

### Environments

**firmware** - Стандартна ESP32
**firmware_esp32s3** - ESP32-S3
**firmware_esp32c3** - ESP32-C3

### Компресія веб-ресурсів

Веб-ресурси автоматично стискаються перед збіркою завдяки `tools/compress_assets.py`:
- Скрипт запускається автоматично при збірці
- Генерує `src/web_assets.h` з стиснутими даними
- Зменшення розміру на **73.7%**

Детальніше: [WEB_ASSETS_README.md](WEB_ASSETS_README.md)

## 🔌 Схеми підключення

### Основні піни (за замовчуванням для JAAM)

```cpp
// LED стрічки
#define MAIN_LED_PIN      13    // Основна стрічка (мапа)
#define BG_LED_PIN        12    // Фонова підсвітка
#define SERVICE_LED_PIN   25    // Сервісні LED

// Кнопки
#define BUTTON_1_PIN      5     // Кнопка 1 (JAAM 3)
#define BUTTON_2_PIN      0     // Кнопка 2
#define BUTTON_3_PIN      2     // Кнопка 3

// Звук
#define BUZZER_PIN        33    // Buzzer
#define DF_RX_PIN         17    // DFPlayer RX
#define DF_TX_PIN         16    // DFPlayer TX

```

### Підключення дисплея (I2C)
- VCC → 3.3V
- GND → GND
- SDA → GPIO 21
- SCL → GPIO 22

### Підключення сенсорів

**BME280/BMP280 (I2C):**
- VCC → 3.3V
- GND → GND
- SDA → GPIO 21
- SCL → GPIO 22

**BH1750 (I2C):**
- VCC → 3.3V
- GND → GND
- SDA → GPIO 21
- SCL → GPIO 22

## 📝 Налаштування збереження

Всі налаштування зберігаються у внутрішній флеш-пам'яті ESP32 (Preferences/NVS):
- Відновлюються після перезавантаження
- Зберігаються після перепрошивки

## 🔄 Оновлення прошивки

### Через веб-інтерфейс (OTA)
1. Відкрийте веб-інтерфейс
2. Перейдіть в розділ "Оновлення прошивки"
3. Оберіть канал (Production/Beta)
4. Оберіть версію зі списку
5. Натисніть "Підтвердити" для початку оновлення

### Через USB
```bash
platformio run -e firmware --target upload
```

### Через [FLASHER](https://flasher.jaam.net.ua/)
1. Відкрийте браузер Chrome/Edge
2. Перейдіть на https://flasher.jaam.net.ua/
3. Підключіть ESP32 через USB
4. Оберіть версію та встановіть

## 📊 Технічні характеристики

### Системні вимоги
- **MCU**: ESP32 (240MHz dual-core)
- **Flash**: 4MB мінімум
- **RAM**: 520KB (heap 200KB+)
- **WiFi**: 2.4GHz 802.11 b/g/n

### Споживання енергії
- **JAAM 1.3/2.1**: ~2-5W (залежно від яскравості)
- **JAAM 3.2**: ~3-10W (залежно від яскравості)

### Підтримувані LED
- WS2812B
- WS2812B-2020 (компактна версія)
- Інші сумісні з протоколом NeoPixel
