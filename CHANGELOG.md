# 🧪 Beta Реліз 5.0-b9

> ⚠️ **Увага**: Це бета-версія для тестування. Може містити нестабільні функції.

## 🆕 Що нового у версії 5.0-b9

### 🔧 Технічні покращення
- **Надійність відображення тривог**: Оптимізовано логіку автоматичного переходу (fallback) між районами та областями. Це забезпечує стабільнішу роботу мапи у випадках, коли дані про конкретний район тимчасово недоступні.
- **Покращення опису оновлень**: Впроваджено вдосконалену систему генерації списку змін для кращої інформативності майбутніх релізів.

### ⚡ Оптимізація
- **Швидкість веб-інтерфейсу**: Оптимізовано завантаження розділу з історією оновлень. Тепер відображаються лише останні 10 записів, що дозволяє сторінці налаштувань відкриватися швидше.

## 📋 Зміни з попередніх beta версій (відносно версії 4.4)

### 🚀 Нові можливості
- **Перехід на райони**: Повна підтримка відображення тривог за районами замість областей з можливістю власного мапінгу світлодіодів.
- **Оновлення по повітрю (OTA)**: Можливість оновлювати прошивку безпосередньо через веб-інтерфейс, обираючи версії з GitHub.
- **Розширена підтримка заліза**: Додано підтримку контролерів ESP32-C3, S3 (включаючи Super Mini), дисплеїв (OLED/LCD), зумерів для різних типів загроз та зовнішніх сирен.
- **Розумні датчики**: Інтеграція датчиків освітленості (BH1750) для автояскравості та сенсорів клімату (температура/вологість).
- **Home Assistant**: Пряма інтеграція без необхідності використання MQTT-брокера.
- **Функції комфорту**: Хвилина мовчання, нічний режим, кастомізація шрифтів годинника та імпорт/експорт налаштувань.

### 🎨 UI/UX покращення
- **Новий веб-інтерфейс**: Повністю перероблений дизайн (Material Design) з підтримкою темної теми та автоматичним збереженням налаштувань без перезавантаження.
- **Інтерактивність**: Попередній перегляд анімацій при налаштуванні, зручні "color pickers" та панель системних логів у реальному часі.

### ⚡ Оптимізація
- **Швидкодія**: Використання бінарних диференційних пакетів (Websocket-diffs) зменшило обсяг переданих даних у десятки разів.
- **Плавність**: Нові алгоритми масштабування кольорів для максимально м'яких переходів світлодіодів та стиснення ресурсів інтерфейсу для миттєвого завантаження.

### 🐛 Виправлення помилок
- **Стабільність**: Покращено логіку перепідключення до Wi-Fi та WebSocket, виправлено роботу системного "вартового" (watchdog) та усунуто помилки ініціалізації периферії на нових типах плат.
