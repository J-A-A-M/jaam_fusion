# Швидкий старт

Цей посібник допоможе вам швидко почати роботу з JAAM Fusion.

## Що вам знадобиться

### Апаратне забезпечення

**ESP32 мікроконтролер** (виберіть один):

- **ESP32** (класичний, рекомендовано ESP32-DevKitC) — цілком функціональний
- **ESP32-C3** — компактніший, менше енергозатратний
- **ESP32-S3** — найпотужніший, двоядерний, більше пам'яті

**Додаткові компоненти:**

- **LED стрічка** WS2812B або аналогічна (адресна RGB)
- **Блок живлення** 5V (залежить від кількості LED)
- (Опціонально) **OLED дисплей** SSD1306 128x64
- (Опціонально) **Сенсори**: BME280 (температура/вологість), BH1750 (освітленість)

### Програмне забезпечення

Вам не потрібно нічого встановлювати! Використовуйте [FLASHER](https://flasher.jaam.net.ua/) для прошивки прямо з браузера.

## Покрокова інструкція

### Крок 1: Прошивка ESP32

1. Відкрийте [FLASHER](https://flasher.jaam.net.ua/)
2. Підключіть ESP32 до комп'ютера через USB
3. Натисніть кнопку "Connect" і виберіть ваш порт
4. Натисніть "Install" та дочекайтеся завершення

!!! tip "Порада"
    Переконайтеся, що використовуєте якісний USB-кабель з підтримкою передачі даних (не лише зарядний).

### Крок 2: Підключення до WiFi

1. Після прошивки ESP32 створить власну WiFi мережу `JAAM_XXXXXX`
2. Підключіться до цієї мережи (без пароля)
3. Відкрийте браузер та перейдіть на `http://192.168.4.1`
4. Введіть дані вашої домашньої WiFi мережі
5. Після перезавантаження пристрій підключиться до вашої мережі

### Крок 3: Доступ до веб-інтерфейсу

Після підключення до вашої WiFi мережі:

1. Знайдіть IP-адресу пристрою (можна подивитись в роутері або використати mDNS: `http://jaam_xxxxxx.local`, де XXXXXX - те ж саме, що в назві WiFi мережі)
2. Відкрийте веб-інтерфейс в браузері
3. Ви побачите панель налаштувань JAAM Fusion

### Крок 4: Базові налаштування

1. **Часовий пояс**: Встановіть ваш часовий пояс (для України: Europe/Kyiv)
2. **Яскравість**: Налаштуйте яскравість LED (рекомендовано 50-70%)
3. **Режим відображення**: Виберіть бажаний режим (наприклад, "Повітряні тривоги")

### Крок 5: Налаштування LED карти

!!! info "Необов'язковий крок"
    Налаштування LED карти є опціональним! Пристрій поставляється з передвстановленими розкладками для найпопулярніших конфігурацій. Якщо одна з них вам підходить, можете пропустити цей крок.

#### Варіант A: Використовується передвстановлена розкладка (рекомендовано для початку)

Виберіть одну з доступних розкладок у веб-інтерфейсі:

```
Загальні → Режим прошивки
```

| Розкладка | LED | Деталі |
|-----------|-----|--------|
| **Плата JAAM 1.3** | 26 | Перша версія JAAM |
| **Плата JAAM 2.1** | 26 | Друга версія JAAM |
| **Плата JAAM 3.2** | 405 | Розширена карта регіонів України (169 регіонів) |
| **Початок на Одещині + Київ** | 26 | Карта з Одеською областю та Києвом |
| **Початок на Одещині** | 25 | Карта з Одеською областю без Києва |
| **Початок на Закарпатті + Київ** | 26 | Карта з Закарпатською областю та Києвом |
| **Початок на Закарпатті** | 25 | Карта з Закарпатською областю без Києва |
| **Власна карта LED** | - | Для користувацької конфігурації |

**Як вибрати розкладку:**

1. Перейдіть до розділу **Загальні**
2. Знайдіть налаштування **Режим прошивки**
3. Виберіть потрібну розкладку зі списку
4. Збережіть конфігурацію

#### Варіант B: Користувацька LED карта

Якщо ваша конфігурація не збігається з передвстановленими розкладками:

1. Перейдіть до розділу "Map Editor"
2. Вкажіть загальну кількість LED на вашій стрічці
3. Прив'яжіть кожен LED до відповідного регіону України
4. Збережіть конфігурацію

Детальніше про [налаштування LED карти](../hardware/led-mapping.md).

## Поширені проблеми

### Пристрій не створює WiFi мережу

- Перевірте живлення ESP32
- Спробуйте прошити ще раз
- Перевірте, чи підтримується ваша плата

### Не можу підключитися до веб-інтерфейсу

- Переконайтеся, що ESP32 та ваш комп'ютер в одній мережі
- Спробуйте скинути налаштування WiFi (затисніть кнопку Boot на 5 секунд)
- Перевірте IP-адресу в роутері

Більше рішень в розділі [Troubleshooting](../reference/troubleshooting.md).
