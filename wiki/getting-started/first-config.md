# Перша конфігурація

Після встановлення прошивки необхідно виконати початкову налаштування.

## Підключення до WiFi

### Режим Access Point

При першому запуску пристрій створює власну WiFi мережу:

- **SSID**: `JAAM_XXXXXX` (де XXXXXX - частина MAC адреси)
- **Пароль**: Немає (відкрита мережа)

### Налаштування WiFi

1. Підключіться до мережи `JAAM_XXXXXX` (без пароля)
2. Відкрийте `http://192.168.4.1`
3. Перейдіть в "WiFi Settings"
4. Введіть SSID та пароль вашої мережі
5. Збережіть та перезавантажте пристрій

## Базові налаштування

### Часовий пояс

```
Settings → General → Timezone: Europe/Kyiv
```

### Яскравість LED

```
Settings → Display → Brightness: 50-70%
```

### Режим роботи

Виберіть основний режим відображення:

- **Air Alerts** - повітряні тривоги (за замовчуванням)
- **Weather** - погода
- **Desk Lamp** - настільна лампа

## Налаштування LED карти

!!! info "Необов'язковий крок"
    Пристрій поставляється з передвстановленими LED розкладками. Вибір розкладки необхідний для правильної роботи, але ви можете просто вибрати одну з готових сценаріїв на місце того, щоб створювати власну карту.

### Доступні передвстановлені розкладки

| Розкладка | LED | Регіонів | Опис |
|-----------|-----|---------|------|
| **Плата JAAM 1.3** | 26 | 167 | Перша версія JAAM |
| **Плата JAAM 2.1** | 26 | 167 | Друга версія JAAM |
| **Плата JAAM 3.2** | 405 | 169 | Розширена карта України |
| **Початок на Одещині + Київ** | 26 | 167 | Одеська область + Київ |
| **Початок на Одещині** | 25 | 166 | Одеська область |
| **Початок на Закарпатті + Київ** | 26 | 167 | Закарпаття + Київ |
| **Початок на Закарпатті** | 25 | 166 | Закарпаття |
| **Власна карта LED** | - | - | Користувацька конфігурація |

### Як налаштувати LED карту

1. Перейдіть до розділу **Загальні**
2. Виберіть **Режим прошивки**
3. Оберіть потрібну розкладку
4. Збережіть налаштування

### Користувацька LED карта

Якщо жодна предвстановлена розкладка не підходить, ви можете створити власну. Детальніше: [LED Mapping](../hardware/led-mapping.md)
