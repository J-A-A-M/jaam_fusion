# Встановлення

Детальна інструкція з встановлення прошивки JAAM Fusion на ESP32.

## Методи встановлення

### 1. Web Flasher (Рекомендовано)

Найпростіший спосіб - використання онлайн-флешера:

1. Відкрийте [FLASHER](https://flasher.jaam.net.ua/)
2. Підключіть ESP32 до комп'ютера
3. Натисніть "Connect" та виберіть порт
4. Виберіть версію прошивки
5. Натисніть "Install"

### 2. PlatformIO (Для розробників)

Для збірки з вихідного коду:

```bash
# Клонувати репозиторій
git clone https://github.com/J-A-A-M/jaam_fusion.git
cd jaam_fusion

# Встановити залежності (якщо є)
pip install -r requirements.txt

# Зібрати та завантажити
platformio run --target upload
```

Детальніше в розділі [Збірка проєкту](../development/building.md).

## Вимоги

### Апаратні вимоги

**Підтримувані ESP32 чіпи:**

| Чіп | Ядра | Flash | RAM | Рекомендація |
|-----|------|-------|-----|--------------|
| **ESP32** | 2 | 4MB+ | 520KB | ✅ Повна функціональність |
| **ESP32-C3** | 1 | 4MB+ | 400KB | ✅ Компактний, менш енергозатратний |
| **ESP32-S3** | 2 | 8MB+ | 512KB | ✅ Найпотужніший варіант |

**Додаткові компоненти:**

- USB-кабель з підтримкою передачі даних
- Блок живлення 5V/2A+ (залежно від кількості LED)
