# Радіація

Секція **"Радіація"** містить налаштування режиму мапи **"Радіація"** (радіаційний фон по регіонах, у нЗв/год).

Загальний опис самого режиму: [Радіація](../../features/radiation.md).

!!! info
    Дані надходять із сервера. Ця секція **не змінює дані**, а лише задає, **як значення перетворюється на колір** і яким кольором показувати регіони без даних.

## Налаштування рівнів радіації

### Верхній рівень радіації

Перейдіть: **Радіація → Верхній рівень радіації**

Тип: **Слайдер**

Діапазон у UI: **100…2000 нЗв/год** (крок **10**).

Значення за замовчуванням: **2000 нЗв/год**.

**Що це налаштування робить:**

Задає **верхню межу** колірної шкали. Значення розподіляються градієнтом між фіксованим нижнім рівнем `50` нЗв/год і цим верхнім рівнем:

- бірюзовий (`#1cafaf`) — нижній рівень
- пурпуровий (`#ff00ff`) — верхній рівень

Значення вище за верхній рівень показуються кольором верхньої межі; значення нижче `50` — кольором нижньої межі.

!!! note
    Цей клампінг застосовується **лише для кольору**. На дисплеї завжди показується **реальне** значення в нЗв/год.

!!! tip
    Зменшуйте верхній рівень, щоб зробити мапу контрастнішою за звичайних (низьких) значень фону. Після зміни кольори **перераховуються одразу**.

## Колір регіону без даних

### Колір регіону без даних

Перейдіть: **Радіація → Колір регіону без даних**

Тип: **Колір**

Колір за замовчуванням: `#FFFFFF` (білий).

**Що це налаштування робить:**

Задає колір регіонів, для яких немає даних радіаційного фону. За замовчуванням такі регіони не світяться; задайте видимий колір, якщо хочете бачити їх на мапі.
