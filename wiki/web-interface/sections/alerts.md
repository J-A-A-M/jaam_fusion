# Тривоги

Секція **"Тривоги"** керує тим, **які типи загроз** відображати на мапі та використовувати для анімацій/яскравості/звуку.

У прошивці кожна подія має “тип” (внутрішньо — біт). Коли в регіоні одночасно активні кілька подій, прошивка вибирає **найпріоритетнішу з дозволених** і саме її показує.

## Перемикачі типів загроз

Перейдіть: **Тривоги → ...**

У цьому розділі є перемикачі для таких подій:

- **Загроза КАБ**
- **Загроза крилатих та авіаційних ракет**
- **Загроза ударних БПЛА**
- **Розвідувальні БПЛА**
- **Загроза балістичних ракет**
- **Вибухи**

Якщо вимкнути певний тип — він:

- не буде обраний як “поточний” для відображення в регіоні
- не буде використовуватись для відповідних анімацій/яскравості/звуку

!!! note
    Базова подія **"Тривога"** (початок/відбій тривоги) завжди доступна і не вимикається цими перемикачами.

## Як це впливає на мапу

У режимі мапи **"Тривоги"** прошивка для кожного регіону визначає найпріоритетніший активний тип події.

Якщо найпріоритетніша подія **вимкнена** перемикачем — вона пропускається, і прошивка намагається взяти наступну за пріоритетом (яка увімкнена).

## Як це впливає на інші секції

Ці перемикачі також керують тим, чи з’являються налаштування для подій у інших секціях:

- **Анімації** — тип/тривалість/цикл і колір для події
- **Яскравість** — повзунок яскравості для події
- **Звук** — перемикач/мелодія для події

Тобто якщо, наприклад, вимкнути **"Загроза КАБ"**, то в секціях **"Анімації"** та **"Яскравість"** відповідні поля можуть зникнути.

## Рекомендації

- Якщо хочете спростити відображення — вимкніть ті типи, які вам не потрібні.
- Якщо хочете, щоб завжди було видно “звичайну тривогу”, навіть коли паралельно йдуть загрози — вимикайте конкретні загрози, які “перекривають” відображення.
