# Енергосистема

Секція **"Енергосистема"** містить налаштування кольорів для режиму мапи **"Енергосистема"** (статус електропостачання по регіонах).

Загальний опис самого режиму: [Стан електромережі](../../features/power-grid.md).

!!! info
    Дані про стан електромережі надходять із сервера. Ця секція **не змінює дані**, а лише задає, **яким кольором** показувати кожен статус на LED‑мапі.

## Кольори статусів енергосистеми

Кожному статусу можна задати власний колір. Після зміни мапа перераховується одразу.

### Достатньо

Перейдіть: **Енергосистема → Достатньо**

Колір регіонів, де електропостачання в нормі.

Колір за замовчуванням: 🟢 `#00FF00` (зелений).

### Не вистачає

Перейдіть: **Енергосистема → Не вистачає**

Колір регіонів з дефіцитом потужності.

Колір за замовчуванням: 🟡 `#FFFF00` (жовтий).

### Значно не вистачає

Перейдіть: **Енергосистема → Значно не вистачає**

Колір регіонів із суттєвим дефіцитом.

Колір за замовчуванням: 🟠 `#FF8C00` (помаранчевий).

### Відключення

Перейдіть: **Енергосистема → Відключення**

Колір регіонів з відключенням електроенергії.

Колір за замовчуванням: 🔴 `#FF0000` (червоний).

### Невідомий

Перейдіть: **Енергосистема → Невідомий**

Колір регіонів, для яких статус не визначено **або немає даних**.

Колір за замовчуванням: ⚪ `#FFFFFF` (білий).

!!! note "Чому "Невідомий" важливий"
    Регіони без даних або з невідомим статусом завжди показуються цим кольором, а не кольором "Достатньо". Так мапа не вводить в оману виглядом "все добре", коли даних насправді немає.

## Як кольори застосовуються на мапі

- Якщо одна LED належить кільком регіонам — перемагає **найгірший статус** (Відключення → Значно не вистачає → Не вистачає → Достатньо → Невідомий).
- Докладніше про логіку відображення: [Стан електромережі](../../features/power-grid.md).
