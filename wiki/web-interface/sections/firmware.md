# Оновлення прошивки

Секція **"Оновлення прошивки"** дозволяє:

- отримувати повідомлення про доступне оновлення
- вибирати канал оновлень (**Prod** або **Beta**)
- запустити оновлення до конкретної версії (з підтвердженням)

!!! important
    Оновлення завантажується через Wi‑Fi та після успіху **пристрій перезавантажиться**.
    Не вимикайте живлення під час процесу.

## Показувати повідомлення про нову прошивку

Перейдіть: **Оновлення прошивки → Показувати повідомлення про нову прошивку**

Якщо увімкнено і прошивка бачить, що у вибраному каналі є **новіша версія**, пристрій показує повідомлення на дисплеї.

Додатково:

- якщо кнопки **не налаштовані**, повідомлення може показувати підказку з IP адресою (щоб зайти у веб‑інтерфейс)
- якщо кнопки **налаштовані**, повідомлення може показувати підказку "натисніть та тримайте кнопку" для запуску оновлення

!!! note
    Якщо в момент доступного оновлення кнопки активні та дисплей увімкнений, **довге натискання кнопки** запускає оновлення (навіть якщо для цієї кнопки налаштована інша дія).

## Канал оновлення прошивки

Перейдіть: **Оновлення прошивки → Канал оновлення прошивки**

Доступні варіанти:

- **Prod** — стабільні релізи
- **Beta** — тестові збірки

Канал впливає на:

- те, з якого списку береться поле **"Прошивка"**
- те, для якого каналу прошивка визначає, чи є "доступне оновлення"

!!! tip
    Якщо ви не тестуєте нові збірки навмисно — залишайте **Prod**.

## Вибір версії прошивки (з підтвердженням)

Перейдіть: **Оновлення прошивки → Прошивка**

Це поле має тип "вибір зі списку + підтвердження":

1. Оберіть потрібну версію зі списку.
2. Перегляньте **Release notes** (веб‑інтерфейс підвантажує опис релізу з GitHub за тегом версії).
3. Натисніть кнопку **"Оновити"** і підтвердіть дію.

Після підтвердження веб‑інтерфейс відправляє вибрану версію в прошивку, і оновлення стартує майже одразу.

!!! warning
    Якщо список версій ще не завантажився (наприклад, щойно увімкнули пристрій і немає зв’язку), прошивка може відхилити запит як **"Невідома версія"**.
    Зачекайте, поки пристрій під’єднається та отримає список доступних версій.

## Як проходить оновлення

Під час оновлення прошивка:

- очищає активні анімації
- завантажує пакет оновлення з сервера `update.jaam.net.ua`
- показує прогрес на дисплеї у відсотках
- передає прогрес у Home Assistant (якщо налаштовано)

Після успішного завершення пристрій перезавантажується.

!!! note
    У прошивці увімкнено показ типових помилок оновлення на дисплеї (наприклад, "замало місця", "невідомий розмір файлу", тощо).
