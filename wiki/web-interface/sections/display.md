# Дисплей

Секція **"Дисплей"** керує параметрами OLED-дисплея (якщо він встановлений) та тим, **що саме** і **коли** показується на екрані.

!!! note
    Підключення дисплея (дроти/I2C) описано окремо: [OLED Дисплей](../../hardware/oled-display.md)

## Тип і параметри OLED-дисплея

Ці налаштування можуть бути недоступні для деяких апаратних профілів (залежить від **"Загальні → Режим прошивки"**).

### Тип дисплею

Перейдіть: **Дисплей → Тип дисплею**

Доступні значення:

- **Відключено**
- **SSD1306**
- **SH1106G**
- **SH1107**

Якщо OLED-дисплей не встановлено — оберіть **"Відключено"**.

### Висота дисплею

Перейдіть: **Дисплей → Висота дисплею**

Варіанти: **32** або **64** (пікселі по висоті).

!!! tip
    Типові модулі SSD1306 бувають 128×32 або 128×64 — виставляйте відповідно до вашого екрана.

### Поворот дисплею

Перейдіть: **Дисплей → Поворот дисплею**

Варіанти: **0°**, **90°**, **180°**, **270°**.

## Відображення на екрані

### Інвертувати дисплей

Перейдіть: **Дисплей → Інвертувати дисплей**

Інвертує кольори/пікселі (корисно, якщо ваш модуль має іншу полярність або потрібно змінити контрастність сприйняття).

### Режим дисплею

Перейдіть: **Дисплей → Режим дисплею**

Доступні режими, які підтримуються прошивкою:

- **Вимкнено** — екран вимкнений (службові повідомлення можуть з’являтися, якщо вони активні)
- **Годинник**
- **Погода**
- **Технічна інформація**
- **Мікроклімат**
- **Комбінований**

!!! note
    Деякі додаткові режими можуть бути приховані або недоступні залежно від конфігурації/збірки.

## Комбінований режим

У режимі **"Комбінований"** дисплей автоматично перемикає екрани по колу.

Перейдіть: **Дисплей → Режим дисплею → Комбінований**

### Тривалість відображення одного екрану у Комбінованому режимі

Визначає, скільки секунд показувати кожен екран перед перемиканням.

### Показувати погоду у Комбінованому режимі

Якщо увімкнено — у коло додасться екран **"Погода"**.

### Відображати мікроклімат у Комбінованому режимі (якщо сенсор встановлено)

Якщо увімкнено і кліматичний сенсор доступний — у коло додасться екран **"Мікроклімат"**.

!!! tip
    У комбінованому режимі **"Годинник"** показується завжди, а погода/мікроклімат — лише якщо ви їх увімкнули.

## Нічний режим для дисплея

### Вимикати дисплей вночі

Перейдіть: **Дисплей → Вимикати дисплей вночі**

Якщо увімкнено — дисплей буде очищатися (вимикатися) у нічний час.

!!! note
    Межі дня/ночі задаються в секції **"Загальні"** (параметри **"Початок дня"** та **"Початок ночі"**).

!!! note
    Вимкнення вночі не застосовується, якщо на екрані активне службове повідомлення.

## Годинник

### Шрифт годинника

Перейдіть: **Дисплей → Шрифт годинника**

Параметр доступний у режимах **"Годинник"** та **"Комбінований"**.

Доступні шрифти:

- **Reddit**
- **Victor**
- **M PLUS 1 CODE**
- **Old Standard**
- **DSEG7**
- **Bitcount Grid**

## Сповіщення на екрані

### Тривалість текстових сповіщень на екрані

Перейдіть: **Дисплей → Тривалість текстових сповіщень на екрані**

Визначає, скільки секунд показувати текстові повідомлення (наприклад, службові повідомлення про події/зміни стану) перед поверненням до основного режиму дисплея.
