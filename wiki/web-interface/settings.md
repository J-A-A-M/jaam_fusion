# Налаштування

На цій сторінці описано основні елементи веб-інтерфейсу налаштувань: системну панель, панель тривог, панель логів, а також список секцій налаштувань.

## Системна панель

Системна панель показує актуальну інформацію про стан пристрою. Її можна **згорнути або розгорнути** кнопкою **"Показати/сховати системну панель"** (іконка ⓘ у шапці сторінки).

!!! tip
    Стан панелі (згорнуто/розгорнуто) запам’ятовується в браузері.

Панель відображає такі дані (за наявності відповідного обладнання/функції):

- **Пам'ять** — використано/всього (KB) + індикатор заповнення
- **Процесор** — температура CPU (°C)
- **Версія** — версія прошивки
- **Час роботи** — аптайм пристрою
- **WiFi** — рівень сигналу (dBm)
- **WiFi uptime** — час з моменту підключення до WiFi
- **Websocket** — час з моменту підключення WebSocket
- **Батарея** — напруга (V), якщо увімкнено моніторинг батареї
- **Температура / Вологість / Тиск** — показники кліматичного сенсора (якщо встановлено)
- **Освітленість** — рівень освітлення (lx), якщо встановлено сенсор
- **API клієнтів** — кількість підключених API клієнтів

!!! note
    Якщо дані недоступні, замість метрик може відображатися повідомлення про відсутність системної інформації.

## Панель тривог

Панель тривог показує короткий підсумок і деталізацію активних тривог. Її можна **згорнути або розгорнути** кнопкою **"Показати/сховати панель тривог"** (іконка ⚠ у шапці сторінки).

!!! tip
    Стан панелі (згорнуто/розгорнуто) запам’ятовується в браузері.

У верхній частині панелі є рядок **"Активні тривоги:"**, який показує:

- кількість регіонів з активними тривогами
- загальну кількість активних подій (типів тривог) у цих регіонах

Нижче відображається список регіонів з іконками типів подій. Типи подій, які можуть відображатися:

- **Повітряна тривога**
- **Артобстріл**
- **Вуличні бої**
- **Хімічна загроза**
- **Ядерна загроза**
- **Загроза ударних БПЛА**
- **Загроза крилатих та авіаційних ракет**
- **Загроза КАБ**
- **Загроза балістичних ракет**
- **Вибухи**

Якщо активних тривог немає, панель показує повідомлення **"Немає активних тривог"**.

## Панель логів

Панель логів призначена для діагностики: вона показує поточні логи пристрою у вигляді списку повідомлень. Панель можна **згорнути або розгорнути** кнопкою **"Показати/сховати логи"** (іконка 📄 у шапці сторінки).

!!! tip
    Стан панелі (згорнуто/розгорнуто) запам’ятовується в браузері.

У панелі **"Логи пристрою"** є кнопки:

- **"Показати" / "Зупинити"** — запуск/зупинка оновлення логів
- **✕** — очищення відображених логів у браузері

Коли стрім логів вимкнений, у панелі може відображатися підказка **"Натисніть \"Показати\" для отримання логів"**.

Кожен запис логу містить:

- час (HH:MM:SS)
- тег (наприклад, `[WIFI]`, `[WEBSOCKET]`, `[ALERT]`)
- текст повідомлення

!!! tip
    Кольори тегів допомагають швидше знаходити потрібні підсистеми (WiFi, WebSocket, тривоги, оновлення, сенсори тощо).
