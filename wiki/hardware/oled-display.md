# OLED Display

Підключення та налаштування OLED дисплею до ESP32.

## Технічні характеристики

## Підтримувані дисплеї

| Тип | Розмір | Інтерфейс | I2C Адреси |
|-----|--------|-----------|-----------|
| SSD1306 | 128x32, 128x64 | I2C | 0x3C, 0x3D |
| SH1106G | 128x64 | I2C | 0x3C, 0x3D |
| SH1107 | 128x128 | I2C | 0x3C, 0x3D |

## Підключення до ESP32

### Схема підключення

| Дисплей | ESP32 |
|---------|-------|
| VCC | 3.3V |
| GND | GND |
| SDA | GPIO21 |
| SCL | GPIO22 |

### Примітки

- **Напруга:** Використовуйте 3.3V, НЕ 5V
- **I2C інтерфейс:** ESP32 має 2 I2C інтерфейси (I2C0, I2C1)
- **GPIO:** GPIO21 (SDA) та GPIO22 (SCL) - стандартні пини I2C1
- **Адреса:** Адреса відображається при скануванні I2C

## Налаштування у прошивці

Налаштування типу та розмірів дисплею знаходиться в розділі **Дисплей** веб-інтерфейсу:

```
Дисплей → Тип дисплею
Дисплей → Висота дисплею
```
