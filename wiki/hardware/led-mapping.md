# LED Mapping

Кастомний маппінг LED - одна з ключових особливостей JAAM Fusion 5.x, яка дозволяє налаштувати будь-яку конфігурацію світлодіодів.

## Що таке LED Mapping?

LED Mapping - це прив'язка кожного світлодіода на вашій стрічці до конкретного регіону України. Це дозволяє відображати інформацію про кожну область окремо.

## Редактор карти LED

### Відкриття редактора

1. Перейдіть у веб-інтерфейс пристрою
2. Відкрийте розділ "Map Editor"
3. Ви побачите інтерактивний редактор

### Базові налаштування

#### 1. Загальна кількість LED

Вкажіть скільки світлодіодів на вашій стрічці:

```
Settings → LED Count: [число LED]
```

#### 2. Прив'язка LED до регіонів

Для кожного LED:

1. Виберіть LED за номером (0, 1, 2...)
2. Виберіть регіон зі списку
3. Натисніть "Assign"

### Типові конфігурації

#### Лінійна стрічка (25 LED)

```
LED 0-24 → 25 регіонів України + Київ
```

#### Матриця 5x5

```
Кожен рядок/стовпець відповідає певній області карти
```

## Збереження та перевірка

### Збереження конфігурації

1. Після налаштування всіх LED натисніть "Save"
2. Конфігурація збережеться в EEPROM
3. Після перезавантаження налаштування відновляться

### Перевірка маппінгу

1. Перейдіть в режим "Test Mode"
2. Кожен регіон підсвітиться по черзі
3. Перевірте чи правильно відображаються області

## Поради

!!! tip "Нумерація LED"
    LED нумеруються з 0. Перший LED на стрічці - це LED 0.

!!! tip "Напрямок стрічки"
    Переконайтеся що знаєте з якого боку починається ваша стрічка (Data In).

!!! warning "Регіони без LED"
    Якщо регіон не має прив'язаного LED, він не буде відображатися.

## Приклади конфігурацій

Приклади різних конфігурацій LED-карт будуть додані пізніше.

## Експорт/Імпорт конфігурації

### Експорт

```
Map Editor → Export → Завантажити JSON
```

### Імпорт

```
Map Editor → Import → Вибрати файл → Upload
```

## Troubleshooting

### LED не світяться

- Перевірте підключення стрічки
- Перевірте рівень сигналу (може знадобитися резистор)
- Перевірте живлення

### Неправильні кольори

- Перевірте тип LED в налаштуваннях (RGB vs GRB)
- Перевірте напругу живлення (має бути 5V)

Більше в розділі [Troubleshooting](../reference/troubleshooting.md).
