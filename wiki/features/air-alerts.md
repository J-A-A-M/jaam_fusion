# Повітряні тривоги

Режим відображення повітряних тривог в реальному часі - основна функція JAAM Fusion. Система отримує дані з сервера [jaam.net.ua](http://jaam.net.ua/) та відображає різні типи загроз на LED-карті України.

## Як це працює

Пристрій автоматично підключається до серверу і отримує інформацію про стан тривог у кожному регіоні України в реальному часі. Система підтримує різні типи загроз з індивідуальним кольоровим відображенням.

## Типи тривог

Система підтримує наступні типи загроз з окремою кольоровою індикацією. Кольори налаштовуються в розділі **Анімації**:

| Тип загрози | Назва у веб-інтерфейсі | Дефолтний колір | Опис |
|-------------|------------------------|-----------------|------|
| **Загальна тривога** | Початок тривог | 🔴 Червоний | Повітряна тривога |
| **Дрони** | Загроза ударних БПЛА | 🟣 Маджента | Виявлено ударні дрони |
| **Ракети** | Загроза крилатих та авіаційних ракет | 🟪 Фіолетовий | Ракетна небезпека |
| **КАБи** | Загроза КАБ | 🟡 Жовтий | Керовані авіаційні бомби |
| **Балістика** | Загроза балістичних ракет | ⚪ Білий | Балістичні ракети |
| **Вибухи** | Вибухи | 🔵 Ціан | Зафіксовано вибухи |
| **Розвідувальні дрони** | Розвідувальні БПЛА | 🔷 Синій | Розвідувальні БПЛА |

### Спеціальні кольори стану

Налаштування кольорів знаходяться в розділі **Анімації**:

| Налаштування у веб-інтерфейсі | Дефолтний колір | Призначення |
|-------------------------------|-----------------|-------------|
| **Відбій тривог** | 🟢 Зелений | Регіон без тривоги |
| **Початок тривог** | 🔴 Червоний | Оголошена тривога |
| **Домашній регіон** | 🟢 Світло-зелений | Спеціальний колір для вашого регіону |

!!! note "Налаштування кольорів"
    Всі кольори можна змінити у веб-інтерфейсі в розділі **Анімації**.

## Увімкнення/вимкнення типів загроз

Кожен тип загрози можна окремо увімкнути або вимкнути в розділі **Тривоги**:

```
Тривоги → Загроза КАБ
Тривоги → Загроза крилатих та авіаційних ракет
Тривоги → Загроза ударних БПЛА
Тривоги → Розвідувальні БПЛА
Тривоги → Загроза балістичних ракет
Тривоги → Вибухи
```

**Дефолтно:** Всі типи увімкнені

## Типи анімацій

Для кожного типу тривоги можна налаштувати окрему анімацію:

### Доступні типи анімацій

| Назва | Опис | Рекомендовано для |
|-------|------|-------------------|
| **FADE** | Плавне згасання/розгорання | Загальна тривога, відбій |
| **BLINK** | Швидке блимання | Критичні загрози (балістика) |
| **BLEND_FADE** | Плавний перехід між двома кольорами | Дрони, ракети |
| **PULSE** | М'яка пульсація | Нові тривоги |
| **ONE_WAY_BLEND_FADE** | Односторонній перехід | Перехідні стани |
| **RUNNING_LIGHT** | Біжуче світло | Спеціальні ефекти |
| **SET_BRIGHTNESS** | Статична яскравість | Без анімації |
| **OFF** | Вимкнено | - |
| **COLOR_PULSE** | Кольорова пульсація | Вибухи |
| **COLOR_BLINK** | Кольорове блимання | Екстрені випадки |

### Налаштування анімацій

Розділ **Анімації** дозволяє вибрати тип анімації для кожного виду тривоги:

```
Анімації → Початок тривог
Анімації → Відбій тривог
Анімації → Загроза ударних БПЛА
Анімації → Розвідувальні БПЛА
Анімації → Загроза крилатих та авіаційних ракет
Анімації → Загроза КАБ
Анімації → Загроза балістичних ракет
Анімації → Вибухи
```

### Параметри анімації

Тривалість показу кожного типу анімації налаштовується в розділі **Анімації** → **Налаштування тривалості показу анімацій (в секундах)**:

```
Анімації → Початок тривог (тривалість)
Анімації → Відбій тривог (тривалість)
Анімації → Загроза ударних БПЛА (тривалість)
Анімації → Розвідувальні БПЛА (тривалість)
Анімації → Загроза крилатих та авіаційних ракет (тривалість)
Анімації → Загроза КАБ (тривалість)
Анімації → Загроза балістичних ракет (тривалість)
Анімації → Вибухи (тривалість)
```

**Діапазон:** 5-600 секунд за замовчуванням

!!! warning "Синхронні анімації"
    Увімкніть **Синхронні анімації** в розділі **Анімації**, щоб всі LED мигали одночасно.

## Домашній регіон

Система дозволяє визначити ваш домашній регіон для спеціальної індикації та сповіщень.

### Налаштування

Розділ **Загальні**:

```
Загальні → Домашній регіон
```

**Дефолтний регіон:** Київ

### Особливості домашнього регіону

1. **Спеціальний колір:** Світло-зелений замість стандартного зеленого
2. **Окрема яскравість:** Можна налаштувати окрему яскравість для домашнього регіону

```
Яскравість → Домашній регіон
```

!!! tip "Рекомендація"
    Встановіть яскравість домашнього регіону вищою, щоб він виділявся на карті.

## Яскравість для різних типів

Розділ **Яскравість** дозволяє налаштувати яскравість для кожного типу тривоги:

```
Яскравість → Тривога
Яскравість → Без тривоги
Яскравість → Вибухи
Яскравість → Загроза крилатих та авіаційних ракет
Яскравість → Загроза ударних БПЛА
Яскравість → Розвідувальні БПЛА
Яскравість → Загроза КАБ
Яскравість → Загроза балістичних ракет
Яскравість → Домашній регіон
```

**Діапазон:** 0-100% для усіх типів

!!! note "Відносна яскравість"
    Всі значення яскравості є відносними до загальної яскравості пристрою.

## Апаратні сповіщення

### Піни оповіщення

Розділ **Сирена** дозволяє налаштовувати піни оповіщення для двох незалежних пристроїв.

#### Пристрій 1

```
Сирена → Пристрій 1 → Режим роботи
Сирена → Пристрій 1 → Пін тривоги
Сирена → Пристрій 1 → Пін відбою
Сирена → Пристрій 1 → Тривалість активації
Сирена → Пристрій 1 → Активний рівень
```

#### Пристрій 2

```
Сирена → Пристрій 2 → Режим роботи
Сирена → Пристрій 2 → Пін тривоги
Сирена → Пристрій 2 → Пін відбою
Сирена → Пристрій 2 → Тривалість активації
Сирена → Пристрій 2 → Активний рівень
```

### Режими роботи пінів

| Режим | Опис |
|-------|------|
| **Бістабільний режим** | Пін змінює та утримує стан (HIGH при тривозі, LOW при відбої) |
| **Імпульсний режим** | Короткий імпульс при активації, з тривалістю, що налаштовується |

### Активний рівень

- **HIGH (логічна одиниця)**: Пін встановлюється в HIGH при тривозі
- **LOW (логічний нуль)**: Пін встановлюється в LOW при тривозі (інверсна логіка)

!!! example "Підключення сирени"
    ```
    Сирена → Пристрій 1 → Пін тривоги: GPIO 25
    Сирена → Пристрій 1 → Режим роботи: Бістабільний режим
    Сирена → Пристрій 1 → Активний рівень: HIGH
    
    При тривозі у домашньому регіоні → GPIO 25 встановлюється в HIGH → Сирена активна
    При відбої у домашньому регіоні → GPIO 25 встановлюється в LOW → Сирена вимкнена
    ```

## Troubleshooting

### Дані не оновлюються

**Діагностика:**

1. Перевірте статус підключення в системній панелі вебінтерфейсу
2. Перевірте логи в системній панелі вебінтерфейсу
3. Перевірте з'єднання з інтернетом

**Рішення:**

- Перезавантажте пристрій
- Перевірте підключення до WiFi
- Перевірте налаштування мережі

### Неправильні кольори

1. Перевірте [LED Mapping](../hardware/led-mapping.md) - чи правильно прив'язані LED до регіонів
2. Перевірте порядок кольорів LED стрічки в налаштуваннях апаратного забезпечення
3. Налаштуйте кольори у розділі **Анімації**

### LED не відображають тривоги

**Чеклист:**

- [ ] Режим карти встановлено на "Тривоги"
- [ ] Пристрій підключено до інтернету
- [ ] LED Mapping налаштовано
- [ ] Яскравість не встановлена на 0%
- [ ] Час синхронізовано

### Проблеми з апаратними сповіщеннями

1. Перевірте правильність номерів пінів
2. Перевірте режим (Бістабільний рівень/Імпульсний режим)
3. Перевірте активний рівень (HIGH/LOW)
4. Перевірте підключення пристрою оповіщення
