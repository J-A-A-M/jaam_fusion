# Радіація

**Радіація** — режим відображення радіаційного фону по регіонах України (в нЗв/год).

Він працює у двох місцях:

- **на LED‑мапі** (режим мапи **"Радіація"**)
- **на дисплеї** (режим дисплею **"Радіація"** або як частина **"Комбінованого"** режиму)

## Що потрібно, щоб все працювало

- Пристрій підключений до Wi‑Fi та має доступ до інтернету.
- Обрано режим мапи **"Радіація"** або режим дисплею **"Радіація"** (залежно від того, що саме ви хочете бачити).

Дані надходять із сервера. Доки сервер не надіслав значення для регіону, цей регіон вважається таким, що **не має даних** (див. нижче).

## LED‑мапа: режим "Радіація"

Перейдіть: **Загальні → Режим мапи → Радіація**.

У цьому режимі кожен регіон підсвічується кольором, що відповідає рівню радіаційного фону.

### Як читати кольори

Колір будується за градієнтом залежно від значення:

- **нижчий рівень** → бірюзовий (`#1cafaf`)
- **вищий рівень** → пурпуровий (`#ff00ff`)

Чим вище значення радіаційного фону — тим ближче колір до верхньої межі градієнта.

### Шкала значень

- **Нижній рівень шкали — фіксований: `50` нЗв/год.** Значення нижче `50` показуються кольором нижньої межі градієнта.
- **Верхній рівень шкали — налаштовується** параметром **Верхній рівень радіації** (за замовчуванням `2000` нЗв/год, діапазон `100…2000`). Значення вище за нього показуються кольором верхньої межі градієнта.

!!! note "Клампінг лише для кольору"
    Обмеження `[50 … верхній рівень]` застосовується **тільки для розрахунку кольору**. На дисплеї завжди показується **реальне** значення в нЗв/год, навіть якщо воно нижче `50` або вище за верхній рівень.

### Зміна верхнього рівня

Перейдіть: **Радіація → Верхній рівень радіації**.

Після зміни цього параметра кольори **перераховуються одразу** і мапа відображається з новою шкалою. Зменшуйте верхній рівень, щоб зробити мапу контрастнішою за низьких значень фону.

### Колір регіону без даних

Якщо для регіону немає даних, він підсвічується **налаштовуваним кольором** — параметр **Радіація → Колір регіону без даних**.

За замовчуванням цей колір — `#FFFFFF` (білий). Задайте інший колір, якщо хочете.

### Якщо LED належить кільком регіонам

У деяких мапінгах одна й та сама LED може відповідати кільком регіонам (наприклад, у режимах із районами).

У такому випадку пристрій бере **середнє значення** радіаційного фону по цих регіонах.

## Дисплей: значення домашнього регіону

### Режим дисплею "Радіація"

Перейдіть: **Дисплей → Режим дисплею → Радіація**.

На екрані показується реальне значення радіаційного фону **домашнього регіону** у форматі `<значення> нЗв/год`.

Якщо даних немає — відображається `-- нЗв/год`.

Домашній регіон задається в **Загальні → Домашній регіон**.

### Комбінований режим дисплею

Перейдіть:

- **Дисплей → Режим дисплею → Комбінований**
- **Дисплей → Показувати радіацію у Комбінованому режимі**

Тривалість відображення одного екрану в комбінованому режимі задається в:

- **Дисплей → Тривалість відображення одного екрану у Комбінованому режимі**

!!! note
    У комбінованому режимі годинник показується завжди, а радіація — лише якщо увімкнено відповідний перемикач.

## Перемикання кнопками

Якщо до пристрою підключені кнопки і їм призначено дію зміни режиму мапи/дисплею, режим **"Радіація"** бере участь у звичайному перемиканні по колу разом з іншими режимами.

## Troubleshooting

### Мапа не показує радіацію

Перевірте:

- **Загальні → Режим мапи** = **Радіація**
- пристрій підключений до Wi‑Fi та має інтернет

Якщо зв’язок є, але кольори не змінюються — найімовірніше сервер ще не надіслав дані. Подивіться логи.

### На дисплеї показує `-- нЗв/год`

Зазвичай це означає, що для домашнього регіону ще не отримано дані радіаційного фону.
