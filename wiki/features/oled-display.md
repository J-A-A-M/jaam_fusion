# OLED Display - Функціонал

!!! note "Апаратна опція"
    OLED Display — це **опціональна апаратна функція**, доступна тільки на деяких версіях пристрою.
    Налаштування дисплея знаходяться у розділі **Дисплей** веб-інтерфейсу.

## Типи дисплеїв

Розділ **Дисплей** дозволяє вибрати тип підключеного дисплея:

```
Дисплей → Тип дисплею
```

**Підтримувані типи:**

- **Відключено** — дисплей вимкнений
- **SSD1306** — 128x32 або 128x64 дисплей
- **SH1106G** — 128x64 дисплей
- **SH1107** — 128x128 дисплей

## Фізичні налаштування

### Висота дисплею

```
Дисплей → Висота дисплею
```

**Опції:**
- **32 пікселів** — для малих дисплеїв
- **64 пікселів** — стандартна висота

### Поворот дисплею

```
Дисплей → Поворот дисплею
```

**Опції:** 0°, 90°, 180°, 270°

### Інвертування кольорів

```
Дисплей → Інвертувати дисплей
```

Перевертає чорні і білі пікселі на дисплеї (корисно для читабельності за певних умов).

## Режими відображення

Розділ **Дисплей** дозволяє вибрати, що відображати на екрані:

```
Дисплей → Режим дисплею
```

**Доступні режими:**

| Режим | Назва | Опис |
|-------|-------|------|
| **0** | **Вимкнено** | Дисплей не активний |
| **1** | **Годинник** | Показує поточний час та дату |
| **2** | **Погода** | Інформація про погоду (якщо доступна) |
| **3** | **Технічна інформація** | Статус системи, версія прошивки, мережа тощо |
| **4** | **Мікроклімат** | Температура та вологість (якщо підключено сенсор) |
| **5** | **Енергосистема** | Інформація про енергосистему України |
| **6** | **Радіація** | Дані про радіаційний фон |
| **9** | **Комбінований** | Циклічне перемикання між режимами |

## Комбінований режим

При виборі режиму **"Комбінований"** (9) дисплей автоматично перемикається між режимами:

**Налаштування:**

```
Дисплей → Тривалість відображення одного екрану у Комбінованому режимі
```

**Діапазон:** 1-15 секунд

**Додаткові опції для Комбінованого режиму:**

```
Дисплей → Показувати погоду у Комбінованому режимі
Дисплей → Відображати мікроклімат у Комбінованому режимі (якщо сенсор встановлено)
```

## Додаткові налаштування

### Вимикання вночі

```
Дисплей → Вимикати дисплей вночі
```

Автоматично вимикає дисплей у нічні години для економії енергії.

### Шрифт годинника

```
Дисплей → Шрифт годинника
```

Дозволяє вибрати шрифт для режимів з часом (годинник, комбінований).

**Доступні шрифти:** Reddit, Victor, M PLUS 1 CODE

### Тривалість текстових сповіщень

```
Дисплей → Тривалість текстових сповіщень на екрані
```

**Діапазон:** 1-60 секунд

Встановлює час показу текстових сповіщень (наприклад, про несправності або стан системи).
