# Стан електромережі

**Енергосистема** — режим відображення статусу електропостачання по регіонах України.

Він працює у двох місцях:

- **на LED‑мапі** (режим мапи **"Енергосистема"**)
- **на дисплеї** (режим дисплею **"Енергосистема"** або як частина **"Комбінованого"** режиму)

## Що потрібно, щоб все працювало

- Пристрій підключений до Wi‑Fi та має доступ до інтернету.
- Обрано режим мапи **"Енергосистема"** або режим дисплею **"Енергосистема"** (залежно від того, що саме ви хочете бачити).

Дані про стан електромережі надходять із сервера. Доки сервер не надіслав дані для регіону, цей регіон вважається таким, що **не має даних** (див. нижче).

## LED‑мапа: режим "Енергосистема"

Перейдіть: **Загальні → Режим мапи → Енергосистема**.

У цьому режимі кожен регіон підсвічується кольором, що відповідає його **статусу електропостачання**.

### Статуси та кольори

| Статус | Значення | Колір за замовчуванням |
|--------|----------|------------------------|
| **Достатньо** | електропостачання в нормі | 🟢 зелений (`#00FF00`) |
| **Не вистачає** | дефіцит потужності | 🟡 жовтий (`#FFFF00`) |
| **Значно не вистачає** | суттєвий дефіцит | 🟠 помаранчевий (`#FF8C00`) |
| **Відключення** | відключення електроенергії | 🔴 червоний (`#FF0000`) |
| **Невідомий / немає даних** | статус не визначено або дані відсутні | ⚪ білий (`#FFFFFF`) |

Усі ці кольори **можна змінити** у веб‑інтерфейсі — див. секцію [Енергосистема](../web-interface/sections/energy.md).

!!! note "Регіон без даних"
    Якщо для регіону немає даних або прийшов невідомий статус, регіон підсвічується кольором **"Невідомий"**. Тобто мапа ніколи не лишає такий регіон у кольорі "Достатньо" (зеленому) — щоб не вводити в оману.

### Якщо LED належить кільком регіонам

У деяких мапінгах одна й та сама LED може відповідати кільком регіонам (наприклад, у режимах із районами).

У такому випадку перемагає **найгірший статус** серед цих регіонів (worst‑severity‑wins) за порядком тяжкості:

**Відключення → Значно не вистачає → Не вистачає → Достатньо → Невідомий**

Тобто якщо хоч в одному регіоні під цією LED є відключення — LED світитиметься червоним.

## Дисплей: статус домашнього регіону

### Режим дисплею "Енергосистема"

Перейдіть: **Дисплей → Режим дисплею → Енергосистема**.

На екрані показується статус електропостачання **домашнього регіону** (текстом): *Достатньо*, *Не вистачає*, *Значно не вистачає*, *Відключення* або *Невідомий*.

Домашній регіон задається в **Загальні → Домашній регіон**.

### Комбінований режим дисплею

Перейдіть:

- **Дисплей → Режим дисплею → Комбінований**
- **Дисплей → Показувати енергосистему у Комбінованому режимі**

Тривалість відображення одного екрану в комбінованому режимі задається в:

- **Дисплей → Тривалість відображення одного екрану у Комбінованому режимі**

!!! note
    У комбінованому режимі годинник показується завжди, а енергосистема — лише якщо увімкнено відповідний перемикач.

## Перемикання кнопками

Якщо до пристрою підключені кнопки і їм призначено дію зміни режиму мапи/дисплею, режим **"Енергосистема"** бере участь у звичайному перемиканні по колу разом з іншими режимами.

## Налаштування кольорів

Кольори статусів винесено в окрему секцію веб‑інтерфейсу:

**Енергосистема → Кольори статусів енергосистеми**

Там можна окремо задати колір для кожного статусу, зокрема для регіонів без даних. Після зміни кольору мапа перераховується одразу.

Докладніше: [Енергосистема (секція налаштувань)](../web-interface/sections/energy.md).

## Troubleshooting

### Мапа не показує статуси (всі регіони "Невідомий")

Перевірте:

- **Загальні → Режим мапи** = **Енергосистема**
- пристрій підключений до Wi‑Fi та має інтернет

Якщо зв’язок є, але всі регіони у кольорі "Невідомий" — найімовірніше сервер ще не надіслав дані стану електромережі. Зачекайте кілька хвилин і подивіться логи.

### На дисплеї показує "Невідомий"

Зазвичай це означає, що для домашнього регіону ще не отримано дані стану електромережі.
