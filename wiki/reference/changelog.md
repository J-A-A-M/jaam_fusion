## Основні релізи

### Version 5.x (2025-2026)

**Революційні зміни:**
- ⭐ Кастомний LED маппінг
- 🎨 Новий веб-інтерфейс
- 🏗️ Модульна архітектура
- 📱 Покращена підтримка OLED
- 🔄 WebSocket real-time updates

### Version 4.x (2024)

**Покращення:**
- Підтримка більше типів LED
- MQTT інтеграція
- Оновлення по повітрю (OTA)

### Version 3.x (2023)

**Основні функції:**
- Повітряні тривоги
- Погода
- OLED дисплей
