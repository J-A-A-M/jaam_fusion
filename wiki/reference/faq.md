# FAQ - Часті питання

## Загальні питання

### Що таке JAAM Fusion?

JAAM Fusion - це прошивка для ESP32, яка перетворює LED-стрічку на інформаційну мапу України з відображенням повітряних тривог, погоди та іншої інформації в реальному часі.

### Чи потрібні навички програмування?

Ні! Для базового використання достатньо:
- Прошити ESP32 через веб-флешер
- Налаштувати WiFi
- Підключити LED стрічку

### Яка вартість проєкту?

Проєкт повністю безкоштовний та відкритий (open-source). Вам потрібно лише придбати компоненти (ESP32, LED стрічка, тощо).

## Апаратне забезпечення

### Які ESP32 плати підтримуються?

Практично всі ESP32 плати з мінімум 4MB Flash пам'яті. Детальніше в розділі [Підтримувані плати](boards.md).

### Скільки LED можна підключити?

Теоретично до 1000+ LED, але практично:
- Рекомендовано: 25-100 LED
- Обмеження: пам'ять ESP32 та потужність блоку живлення

### Які LED стрічки підтримуються?

- WS2812B (найпоширеніша)
- WS2811
- SK6812
- APA106

### Чи потрібен окремий блок живлення для LED?

Так, якщо LED більше 10-15 штук. ESP32 не зможе живити велику стрічку.

## Налаштування

### Як змінити яскравість LED?

```
Web Interface → Settings → Display → Brightness
```

### Як отримувати дані без інтернету?

На жаль, це неможливо. Для отримання актуальних даних про тривоги потрібне підключення до інтернету.

### Чи можна використовувати без OLED дисплею?

Так, OLED дисплей опціональний.

## Проблеми

### Пристрій не підключається до WiFi

1. Перевірте правильність SSID та пароля
2. Переконайтеся що роутер працює на 2.4GHz (ESP32 не підтримує 5GHz)
3. Спробуйте скинути налаштування (затисніть кнопку Boot на 5 секунд)

### LED світяться неправильними кольорами

Перевірте тип LED в налаштуваннях:
```
Settings → Hardware → LED Type: WS2812B (GRB) або RGB
```

### Веб-інтерфейс недоступний

1. Перевірте IP-адресу пристрою в роутері
2. Спробуйте через mDNS: `http://jaam_xxxxxx.local` (де XXXXXX - та ж назва WiFi мережи для налаштування)
3. Перезавантажте пристрій
