# JAAM 3.2

На цій сторінці згодом буде розміщено інформацію та опис плати **JAAM 3.2**.

У прошивці **JAAM Fusion** для цієї плати можуть бути доступні апаратні профілі (вибір у **Загальні → Режим прошивки**):

- **"Плата JAAM 3.2"** — відображення з деталізацією по районах/підрегіонах
- **"Плата JAAM 3.2 (по областям)"** — відображення по областях (як на JAAM 2.1)

Дивіться також: [Загальні](../web-interface/sections/general.md).

## Нотатки по платі

- На платі **JAAM 3.2** передбачено фізичний вимикач (світч) буззера. Якщо звуку немає — перевірте його положення.

- [Звук](../web-interface/sections/sound.md)
- [Buzzer](../hardware/sound/buzzer.md)
